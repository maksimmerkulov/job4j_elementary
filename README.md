# job4j_elementary

[![GitHub Action](https://github.com/maksimmerkulov/job4j_elementary/actions/workflows/maven.yml/badge.svg)](https://github.com/maksimmerkulov/job4j_elementary/actions/workflows/maven.yml)

The project contains solutions for the "Basic Syntax" block of the Job4j course.

This project covers:
- Variables
- Conditional operators
- Loops
- Arrays
- Unit testing
- IDEA
- Basics of GIT
