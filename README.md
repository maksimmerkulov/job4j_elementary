## Overview
This repository contains a collection of solutions focusing on core Java programming concepts. The project emphasizes clean code, technical documentation (Javadoc), and rigorous testing.

### Key Concepts Covered
* **Java Fundamentals**: Variable types, arithmetic operations, and basic expressions.
* **Control Structures**: `if-else` branching, ternary operators, and `switch` statements.
* **Looping Constructs**: Practical application of `for`, `while`, and `do-while` loops.
* **Arrays**: Memory management, iteration, and basic algorithms.
* **Unit Testing**: Integration of **JUnit 5** and **AssertJ** for robust code verification.
* **Development Tools**: Professional use of IntelliJ IDEA and Git workflows.

## Engineering Standards
The project follows professional Java development guidelines:
* **Documentation**: Comprehensive **Javadoc** for methods and classes to ensure maintainability.
* **Testing Style**: Modern **JUnit 5** with **AssertJ** (fluent assertions and precision matching).
* **Code Layout**: Strict adherence to encapsulation and clean code principles.

## Tech Stack
* **Language**: Java 17 (LTS)
* **Build System**: Maven
* **Testing**: JUnit 5, AssertJ
* **IDE**: IntelliJ IDEA

## Usage
Ensure you have **Maven** installed. To verify the solutions and run the test suite, use:
```bash
mvn test
```
